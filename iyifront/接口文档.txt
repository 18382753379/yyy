1：/userLogin
desc：用户登录接口
type:post
data:username,password
return:
    1、账户不存在时
      {
        code:0,
        msg:"账户不存在"
      }
    2、密码错误时
      {
        code:1,
        msg:"密码错误"
      }
    3、登录成功之后把用户的id和username返回
      {
        code:2,
        msg:"登录成功",
        user:{
          id,
          username
        }
      }


2、/getDetail
desc:根据商品id得到goods表中的所有字段信息
type:get
params:gid
return:[{id:1,name:....}]


3、/checkusername
